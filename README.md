## NDA — No Docs Allowed

**NDA** (No Docs Allowed) is a lightweight Visual Studio Code extension that folds and hides XML documentation comments, helping you stay focused on what actually matters — the code.

> **Write docs for others.**
>
> **Hide docs for yourself.**

### Why NDA?

Modern codebases are often overloaded with documentation noise.

NDA lets you **opt-in to clarity** by hiding what you don’t need *right now*.

Because sometimes…
**the best documentation is no documentation.**

![nda in action](https://raw.githubusercontent.com/git-hosted/cdn/refs/heads/master/nda/nda-in-action.png)


### Commands

* `NDA: Show XML Doc Comments`
* `NDA: Hide XML Doc Comments`
* `NDA: Toggle XML Doc Comments`


### Perfect for

* C# / .NET developers
* Large enterprise codebases
* Developers who prefer signal over noise
* Anyone tired of scrolling past `/// <summary>`
